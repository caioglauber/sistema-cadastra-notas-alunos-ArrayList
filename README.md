# Nesse pequeno projeto é mostrado como podemos cadastrar os alunos em um vetor usando a Classe ArrayList.
# É perguntado ao usuario quantos alunos ele deve cadastrar. Em seguida, segue com as informações de cada Aluno.
# Para Cadastrar um novo aluno é necessário utilizar uma matricula diferente. O programa não aceita matriculas repeditas.
# Feito isso, pode ser adicionado um ponto extra na nota de cada aluno solicitando uma matricula válida.
# Após a validação da matricula o programa mostrará as informações dos alunos e a nota final com o ponto extra acrescentado a mesma.
